[−][src]Module yatp::queue

The task queues available for the thread pool.

A task queue has two parts: a shared [TaskInjector] and several [LocalQueue]s. Unlike usual MPMC queues, [LocalQueue] is not required to be Sync. The thread pool will use one [LocalQueue] per thread, which make it possible to do extreme optimizations and define complicated data structs.
