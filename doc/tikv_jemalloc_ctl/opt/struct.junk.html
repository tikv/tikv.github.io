pub struct junk;

jemalloc’s junk filling mode.

Requires --enable-fill to have been specified during build configuration.

If set to “alloc”, each byte of uninitialized allocated memory will be set to 0x5a. If set to “free”, each byte of deallocated memory will be set to 0x5a. If set to “true”, both allocated and deallocated memory will be initialized, and if set to “false” junk filling will be disabled. This is intended for debugging and will impact performance negatively.

The default is “false”, unless --enable-debug was specified during build configuration, in which case the default is “true”.

Examples

use tikv_jemalloc_ctl::opt;
let junk = opt::junk::read().unwrap();
println!("junk filling: {}", junk);
