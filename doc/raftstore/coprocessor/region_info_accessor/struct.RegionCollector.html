pub struct RegionCollector {
    regions: HashMap<u64, RegionInfo>,
    region_ranges: BTreeMap<Vec<u8>, u64>,
}

RegionCollector is the place where we hold all region information we collected, and the underlying runner of RegionInfoAccessor. It listens on events sent by the RegionEventListener and keeps information of all regions. Role of each region are also tracked.

fn check_region_range(
    &mut self,
    region: &Region,
    clear_regions_in_range: bool
) -> bool[src]

For all regions whose range overlaps with the given region or region_id is the same as region's, checks whether the given region's epoch is not older than theirs.

Returns false if the given region is stale, which means, at least one region above has newer epoch. If the given region is not stale, all other regions in the collection that overlaps with the given region must be stale. Returns true in this case, and if clear_regions_in_range is true, those out-of-date regions will be removed from the collection.
